# ADB_Manager

ADB 명령어를 활용한 안드로이드 계열 디바이스 도구 모음입니다.

현재 사용 가능한 도구 목록
 * 스크린샷 캡처
 * Scrcpy를 활용한 디바이스 스크린 출력
 * 동영상 녹화
 * 로그 추출 (추출 옵션 선택 가능)
 * (기능 개발 중) 앱 설치
 * S/N, 디바이스 모델명, 디바이스명 클립보드 복사
